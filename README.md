# Maze Solver (DFS / Backtracking) in C++

A robust console application built in C++ that dynamically generates mazes and finds the optimal exit path using the **Depth-First Search (DFS)** and backtracking algorithms.

Developed as part of the intensive algorithmic challenges at **Penguin Academy**.

## Core Concepts Applied
* **Algorithmic Logic:** Depth-First Search (DFS) and Backtracking.
* **Data Structures:** Stacks and 2D Arrays / Matrices.
* **Memory Management:** Efficient pointer handling and matrix traversal in C++.
* **Procedural Generation:** Randomized maze creation ensuring at least one valid path.

## Features
1. **Dynamic Generation:** Creates unique mazes on every execution.
2. **Automated Pathfinding:** Calculates and visualizes the correct path from start to finish.
3. **Console Visualization:** Renders the maze and the solving process directly in the terminal.

## Tech Stack
* **Language:** C++
* **Environment:** Compatible with any standard C++ compiler (g++, clang).

## How to Run
To compile and run this project locally, use the following commands in your terminal:

```bash
git clone https://github.com/aaronbarriospy/Maze-Solver-DFS-CPP.git
cd Maze-Solver-DFS-CPP
g++ main.cpp -o maze_solver
./maze_solver
```

*(Note: Replace `main.cpp` with the actual name of your main file if it's different).*
